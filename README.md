# Reqly Website

The official website and content hub for the Reqly ecosystem.

This project powers the Reqly website, including product information, documentation, public API discovery, guides and future developer resources.

## Features

- Product showcase
- Documentation
- API discovery
- Developer guides
- Release updates
- Ecosystem information

## Related Projects

- Reqly
- Reqly API
- Reqly Design System

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
